# Spamador-de-Whatsapp
Este é um programa que eu desenvolvi com foco comercial para empresas que tem foco em vendas/atendimento via whatsapp, este programa envia mensagens em massa para vários números, é possível anexar planilhas excel neste programa, ou colar os números no programa, ele é bem simples porque é um dos meus primeiros projetos em python, porém funcional.
